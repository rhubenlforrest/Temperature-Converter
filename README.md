# Temperature Converter

A simple Python GUI application that converts temperatures from **Celsius** to **Fahrenheit**. Built with **Tkinter**, this project demonstrates basic GUI programming and user input handling in Python.

## Features
- Enter a temperature in Celsius and convert it to Fahrenheit instantly.
- Clean, easy-to-read interface using Tkinter.
- Beginner-friendly, ideal for learning Python GUI development.

## How to Run
1. Make sure you have **Python 3.x** installed.
2. Save the file `temperature_converter.py` to your computer.
3. Open a terminal/command prompt in the project folder.
4. Run the app:


python temperature_converter.py
